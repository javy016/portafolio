 Mi Portafolio
La Idea Original
Quería crear algo que no solo muestre mi trabajo, sino que cuente mi historia. Un espacio donde el código y la creatividad se encuentren.
Que la navegación sea intuitiva sin instrucciones

- Decisiones Clave Que Tomé
El Dilema del Diseño
¿Minimalista o con personalidad? Terminé yendo por un balance: estructura limpia con toques de color que me representan.

- La Batalla Mobile vs Desktop
Empecé diseñando para mobile primero - fue la mejor decisión. Si funciona bien en móvil, en desktop fluye naturalmente.

- ¿Qué Proyectos Incluir?
No solo los más "exitosos", sino aquellos donde aprendí algo valioso. Cada proyecto tiene una historia detrás.

- Detalles Técnicos Que Me Obsesionaron
Performance
Cero frameworks pesados
CSS puro para las animaciones
Imágenes optimizadas manualmente
Fuentes del sistema para carga instantánea


Cada línea de CSS tiene su razón de ser

JavaScript solo donde realmente agrega valor


- Mostrar Proyectos Viejos
Incluí algunos proyectos antiguos aunque el código no sea perfecto - muestran mi evolución y eso vale más.

- Mantenerlo Simple
La tentación de agregar features era constante. Tuve que preguntarme: "¿Esto realmente mejora la experiencia o solo la complica?"

- Lo Que Aprendí
Un portafolio nunca está terminado, evoluciona contigo

Menos opciones a veces significa mejor experiencia

La consistencia en pequeños detalles hace la diferencia

Testear con usuarios reales te da perspectivas que nunca considerarías
